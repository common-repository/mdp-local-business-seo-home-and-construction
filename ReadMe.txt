MicroData Project Local business SEO Home and Construction plugin will optimize your website with microdata for google, bing and yahoo.

== Description ==

This plugin has been supper ceded by Local Business Seo. All options that were available with this plugin are now available in Local Business Seo.

Local Business Seo can be found here
http://wordpress.org/plugins/mdp-local-business-seo/

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `mdp-local-business-seo-home-and-construction` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What is microdata? =

Microdata is data stored in specific tags that communicates with search engines that tells them what exactly in on your website.

= Will microdata help my website's seo? =

Yes correct microdata can immensly improve your sites seo.
